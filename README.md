# Antigravity Hub v1.0.0 (Stable Edition) 🚀

**Antigravity Hub** — це модульна веб-оркестраційна платформа (SPA), побудована на фреймворку **FastAPI**. У цьому хабі єдиний "диригент" (Orchestrator) динамічно завантажує власні внутрішні мікросервіси та модулі, пропонуючи користувачам преміальний "Cyber-Tactical HUD" інтерфейс.

## 📦 Вбудовані модулі:
1. **WEB-DLP** (YouTube Downloader & Audio Extractor) — Завантаження аудіо та відео контенту за допомогою `yt-dlp` та `ffmpeg`.
2. **MY RSS** (YouTube & RSS Aggregator) — Ваш персональний агрегатор новин та YouTube-каналів.
3. **PDF Magic** — Інструмент для миттєвого очищення PDF файлів від зайвого фону, тіней (ідеально для друку).
4. **Gemini Voice** — Просунута система Text-to-Speech синтезу з підтримкою **Edge-TTS** (на базі Microsoft) та AI-голосів від **Gemini 3.1 Flash**. Підтримує "AI Steering" для налаштування тональності голосу (Context) через інтерактивний Cyber-HUD інтерфейс швидкого вибору емоцій (інверсійні стилізовані кнопки).

## 🛠 Запуск та розгортання

### 1. Системні вимоги:
- **ОС**: macOS / Windows / Linux
- **Python**: `3.12+`
- **Залежності Системи**: Встановлений **FFmpeg** (у macOS доступний через `brew install ffmpeg`).

### 2. Встановлення:
Система має стабільні залежності. Встановіть потрібні пакети Python (віртуальне середовище за бажанням):

```bash
pip install fastapi uvicorn python-multipart jinja2 pydantic edge-tts google-genai python-dotenv
# Додатково для WEB-dlp
pip install yt-dlp
```

### 3. Налаштування середовища:
Для правильної роботи доступу до преміум-голосів, у вас повинен бути ліцензійний або безкоштовний ключ до Google Gemini API. 
1. Створіть файл `.env` за шляхом `apps/Gemini_Voice/.env`.
2. Додайте туди ключ у форматі `GEMINI_API_KEY=ваш_ключ`.

### 4. Запуск сервера:
Запустіть сервер-оркестратор за допомогою uvicorn:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
Відкрийте у браузері [http://localhost:8000/](http://localhost:8000/)
