# Gemini TTS Web Assistant 🚀 (V4.0 Stable — Cyber-HUD Edition)

Сучасний веб-застосунок для інтелектуального перетворення тексту в мову з футуристичним інтерфейсом у стилі **Cyber-Tactical HUD**. Поєднує стабільність **Microsoft Edge TTS** та емоційну глибину **Google Gemini 3.1**.

---

## 🌟 Статус проекту: V4.0 Stable
Проект досяг стабільного стану з фокусом на бездоганну якість синтезу та повне занурення в атмосферу "техно-генія".

**Що нового в V4.0:**
- **Cyber-HUD Interface**: Повний редизайн в стилі "Amber Glow" з тактильною анімацією та неоновими акцентами.
- **Custom UI Components**: Всі елементи управління (dropdowns, inputs) замінені на кастомні HUD-модулі для 100% візуальної цілісності.
- **Audio Context Steering**: Розширені можливості "режисури" озвучки через текстове поле AI Steering.
- **Оптимізована логіка**: Функції автоматичного покращення тексту видалено задля забезпечення абсолютної точності озвучування наданого оригіналу.
- **Стабільність**: Повна підтримка останніх версій `google-genai` SDK.

## ✨ Ключові можливості

### 1. Мульти-двигунова Озвучка
- **Edge TTS (Default)**: Голоси **Остап** та **Поліна**. Найшвидший та безлімітний варіант для щоденного використання.
- **Gemini 3.1 Flash TTS**: Преміальна емоційна озвучка з 8 нейромережами-характерами:
  - *Charon, Puck, Fenrir, Enceladus, Aoede, Kore, Leda, Algieba*.

### 2. Режисура озвучки (Audio Steering) 🆕
Ви можете керувати настроєм ШІ за допомогою поля **Context**:
- **Приклад**: `Premium commercial. Dynamic pacing—starts intrigued, ends punchy.`
- **Теги в тексті**: Використовуйте `[excited]`, `[whispers]`, `*зітхає*` для миттєвої реакції ШІ.
- **Наголоси**: Керування наголосами за допомогою ВЕЛИКИХ літер (Наприклад: `зАмок` vs `замОк`).

## 🚀 Встановлення та Запуск

### 1. Підготовка
- Переконайтеся, що у вас встановлено **Python 3.9+**.
- Обов'язково встановіть **FFmpeg** (необхідно для фінальної збірки аудіо в форматі `.ogg`).
- Отримайте API ключ на [Google AI Studio](https://aistudio.google.com/) та додайте його у файл `.env`:
  ```env
  GEMINI_API_KEY=ваш_ключ
  ```

### 2. Швидкий Старт
- **macOS**: Запустіть файл `Start_Gemini_TTS.command`.
- **Windows**: Запустіть файл `Start_Gemini_TTS_Win.bat`.
- Після запуску перейдіть за адресою: **[http://localhost:8000](http://localhost:8000)**.

## 🔧 Troubleshooting (Вирішення проблем)

> [!WARNING]
> **FFmpeg Not Found**: Якщо аудіо не програється, перевірте встановлення FFmpeg. В macOS: `brew install ffmpeg`.
> 
> **API Limits (Error 500/429)**: Безкоштовний тариф Gemini має ліміт **10 запитів на добу**. Якщо ліміт вичерпано, використовуйте Edge TTS.
> 
> **Key Error**: Перевірте правильність ключа у файлі `.env`. Ключ не повинен містити зайвих пробілів.
